# Binary Search Tree
Implementation of the BST Data Structure

# Features
- **insert(value)** adds a new node with the given value to the BST
- **deleteItem(value)** removes the node with the given value from the BST
- **find(value)** returns the node with the given value in the BST
- **levelOrderForEach(callback)** traverses the BST in level order and calls the callback on each node
- **inOrderForEach(callback)** traverses the BST in in-order and calls the callback on each node
- **preOrderForEach(callback)** traverses the BST in pre-order and calls the callback on each node
- **postOrderForEach(callback)** traverses the BST in post-order and calls the callback on each node
- **height(value)** returns the height of the node with the given value
- **depth(value)** returns the depth of the node with the given value
- **isBalanced()** returns true if the BST is balanced, and false otherwise
- **rebalance()** rebalances the BST to ensure it is balanced
